### 📂 Featured Projects

#### 🧠 [MindNav BCI](https://github.com/Sakib-Dalal/Mind-Nav) (Brain-Computer Interface)
*A hardware-software hybrid BCI streaming neocortex pyramidal neuron EEG signals to control real-time navigation.*
* Owned the **EEG Data Acquisition & Processing GUI** (Tkinter) running a 60 FPS real-time oscilloscope.
* Engineered serial communication protocols to ingest 256 Hz filtered data from BioAmp EXG Pill / Arduino.
* Standardized synchronized visual/audio REST and CLICK stimulus protocols to build clean training datasets.

#### 🛡️ [BCIShield](https://github.com/Vasco1290/BCIShield) (Adversarial EEG Robustness)
*Research evaluating vulnerability of EEG motor imagery classifiers under adversarial attacks.*
* Constructed adversarial white-box attack pipelines (FGSM & PGD) targeting PyTorch classifiers on BCI Competition IV-2a data.
* Benchmarked stabilization of accuracy using adversarial training and input smoothing as defenses.

* Building a FastAPI backend utilizing vector storage, LangChain security agents, and LightGBM classifier metadata.
